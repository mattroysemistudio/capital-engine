# Capital Modeling Engine

A standalone Next.js application serving the Capital Modeling Engine — one master model for entity structure, equity splits, investor returns, and instrument design.

## Features

- Bilingual interface (English/Spanish)
- Currency conversion (USD ↔ MXN at 17.50 rate via Banxico)
- Real-time calculations across all sliders
- Shareable scenarios via URL
- Dark mode design with SEMI brand palette

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deployed on Vercel. Custom domain: capital.semistudio.com
